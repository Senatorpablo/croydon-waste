# Croydon Waste Management

Cleaner streets. Local jobs. Zero to landfill.

A Croydon-born waste management company website. Single-page site with:
- Six Social Value commitments aligned to Croydon Council's SV framework
- Service cards for council/public sector and business/resident services
- Postcode coverage map across all CR postcodes
- Contact form with enquiry type routing
- Scroll animations via Intersection Observer
- Fraunces + Inter typography with cream/green/lime colour palette

## Tech
- Pure HTML/CSS/JS — no framework
- Responsive design (mobile-first)
- Google Fonts (Fraunces display + Inter)
- GitHub Pages ready

## Deploy
Push to `main` → GitHub Pages auto-deploys from root.
